## ğŸ”’ Public vs Private Rules

### Visibility Options

**Public:**
- Visible to all users
- Can be used in any workflow
- Read-only for non-owners
- Can be duplicated

**Private:**
- Visible only to creator
- Can only be used in creator's workflows
- Full control

**Team Only:**
- Visible to team members
- Can be used in team workflows
- Team admins can edit
